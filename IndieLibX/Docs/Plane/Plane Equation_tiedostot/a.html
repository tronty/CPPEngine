                          songho Mod Aruna Kalidoss • 5 months ago

                          (x1,y1,z1) is not a point on the edge of a plane. It can be any point of the line equation. (And, I noted "p" for a point on the line, "v" for the direction vector of the line.)

                          In order to find 2 end points on the plane, you have to use additional line-line intersection algorithm. First, find all edge lines from a plane. For example if it is a triangle connecting ABC vertices, then you get 3 edge lines; AB, BC and CA. Second, perform the line-to-line intersection algorithm 3 times with an edge line and the intersecting line from 2 planes.
